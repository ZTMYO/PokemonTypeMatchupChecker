# 宝可梦属性相克查询

一个简洁的宝可梦属性克制关系查询工具，支持快速查看属性相克倍率和宝可梦搜索。

## 功能特点

- **属性克制表格**：直观展示 18 种属性之间的克制关系
- **宝可梦搜索**：支持中文名、英文名、拼音、拼音首字母和编号搜索
- **双属性攻击**：支持双属性攻击方，分别显示两个属性的伤害倍率
- **双属性防守**：自动计算双属性防守方的综合倍率
- **攻防交换**：一键交换攻击方和防守方属性
- **属性查询宝可梦**：通过属性筛选查找符合条件的宝可梦
- **响应式设计**：完美适配 PC 和移动端

## 使用方法

1. 直接打开 `index.html` 即可使用
2. 点击表格的行列标题快速选择攻击/防守属性（支持多选）
3. 使用下拉菜单选择攻击方和防守方属性
4. 搜索宝可梦自动填充属性信息
5. 点击"属性查宝可梦"按钮，通过属性筛选宝可梦

## 搜索支持

- 中文名：皮卡丘
- 英文名：pikachu
- 拼音：pikaqiu
- 拼音首字母：PKQ
- 编号：0025
