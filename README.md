# 宝可梦属性相克查询

一个简洁的宝可梦属性克制关系查询工具，支持快速查看属性相克倍率和宝可梦搜索。

## 功能特点

- 📊 **属性克制表格**：直观展示 18 种属性之间的克制关系
- 🔍 **宝可梦搜索**：支持中文名、英文名、日文名和编号搜索
- ⚡ **快速查询**：点击表格或使用下拉菜单快速查看倍率
- 🎯 **双属性计算**：自动计算双属性宝可梦的综合倍率
- 📱 **响应式设计**：完美适配 PC 和移动端
- ⌨️ **键盘导航**：支持方向键导航搜索结果

## 使用方法

1. 直接打开 `index.html` 即可使用
2. 点击表格的行列标题快速查看属性克制
3. 使用下拉菜单选择攻击方和防守方属性
4. 搜索宝可梦自动填充属性信息
